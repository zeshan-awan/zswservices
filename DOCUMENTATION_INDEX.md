### If you're stuck...

1. **Check** `QUICK_REFERENCE.md` Troubleshooting table
2. **Read** `SETUP_GUIDE.md` for that section again
3. **Look** at browser console (F12 ‚Üí Console tab)
4. **Check** Firestore console for data
5. **Read** the errors carefully - they usually tell you the problem

### If still stuck...

1. **Clear browser cache** and hard refresh (Ctrl+Shift+R)
2. **Clear node_modules** and reinstall:
   ```bash
   rm -r node_modules
   npm install
   ```
3. **Check** all credentials are exactly correct
4. **Test** in incognito/private window
5. **Re-read** the relevant documentation file
